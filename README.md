# Moon Sport Club 🏆

Moon Sport Club is a modern, premium web application designed for a premier sports club in Cambodia. The project features a cinematic user interface, smooth animations, and a seamless booking experience for gym and multi-sport facilities.

## ✨ Key Improvements (UI/UX 2.0)

- **Modern Dark Theme**: A sleek, dark-surface aesthetic with vibrant orange accents (`#f97316`).
- **Global Design System**: Shared `global.css` with CSS variables for consistent fonts (Inter), colors, and spacing.
- **Frosted Glass Navigation**: A sticky, translucent navbar with animated dropdowns and "contiguous hover" logic for large screens.
- **Cinematic Hero Sections**: Each page features high-quality imagery with gradient overlays and slide-in typography animations.
- **Glassmorphism Components**: Forms and service cards use semi-transparent backgrounds with backdrop blurring.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 📁 Project Structure

- `index.html`: Optimized home page with hero badge and trainer showcases.
- `css/global.css`: The "brain" of the styling — contains all design tokens and shared components (Nav, Footer, Buttons).
- `pages/`:
  - `gymsport.html`: Activity grid for gym enthusiasts.
  - `matchsport.html`: Sports activity listings (Football, Tennis, etc.).
  - `login.html` & `register.html`: Clean, focused authentication flows.
  - `booking.html`: Streamlined facility reservation form.
  - `feedback.html`: Customer inquiry and feedback portal.

## 🛠️ Built With

- **HTML5**: Semantic structure for better SEO and accessibility.
- **Vanilla CSS3**: Modern features like `backdrop-filter`, CSS variables, and keyframe animations.
- **Material Icons**: Clean, recognizable iconography.
- **Inter Font Family**: Modern sans-serif typography for readability.
